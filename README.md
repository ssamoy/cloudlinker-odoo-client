# CloudLinker Print — Odoo Module

Connect Odoo to [CloudLinker](https://cloudlinker.eu) to print documents directly on local printers from the cloud.

## Features
- Quick setup: login or register directly from Odoo settings
- Print invoices, sale orders, pickings, shipping labels, QWeb reports and POS receipts
- Assign default printer per document type
- Manual printer selection via wizard with copy count
- Auto-print on invoice post / picking validation
- Sync clients and printers from CloudLinker with one click
- Download the CloudLinker client (Windows/Linux) from the settings page

## Requirements
A CloudLinker account is required. Free plan available at [cloudlinker.eu/register](https://cloudlinker.eu/register).

## Compatibility
Odoo 18 — Community & Enterprise

## Installation
1. Add this repository to your Odoo instance as a custom addons path
2. Install the **CloudLinker Print** module via Apps
3. Go to **Settings → CloudLinker** and click **Login** or **Create Account** (or enter your Organization ID and API Key manually)
4. Download and install the CloudLinker client on your print machine
5. Click **Sync Printers** to import your printers
6. Configure default printers in **CloudLinker → Document Rules**
