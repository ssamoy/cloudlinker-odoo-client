# CloudLinker Print — Odoo Module

Connect Odoo to [CloudLinker](https://cloudlinker.eu) to print documents directly on local printers from the cloud.

## Features
- Print invoices, pickings, shipping labels, QWeb reports and POS receipts
- Assign default printer per document type
- Manual printer selection via wizard
- Auto-print on invoice post / picking validation
- Sync clients and printers from CloudLinker with one click

## Requirements
A CloudLinker account is required. Free plan available at [cloudlinker.eu/register](https://cloudlinker.eu/register).

## Compatibility
Odoo 16, 17, 18 — Community & Enterprise

## Installation
1. Add this repository to your Odoo instance as a custom addons path
2. Install the **CloudLinker Print** module via Apps
3. Go to **Settings → CloudLinker** and enter your Organization ID and API Key
4. Click **Sync Printers** to import your printers
5. Configure default printers in **CloudLinker → Document Rules**
