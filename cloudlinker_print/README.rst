==================
CloudLinker Print
==================

Connect Odoo to the `CloudLinker <https://cloudlinker.eu>`_ cloud-to-local print service.
Trigger print jobs on local printers directly from Odoo — without VPNs, without exposing
your network, and without print drivers on the server.

Features
--------
* Configure CloudLinker credentials in Settings (Organization ID + API Key)
* Sync clients and printers from CloudLinker with one click
* Assign default printer per document type (invoices, pickings, shipping labels, QWeb reports, POS)
* Manual printer override via wizard at print time
* Optional auto-print on invoice post / picking validation
* Secure one-time PDF token — no credentials exposed to the CloudLinker agent

Requirements
------------
A CloudLinker account is required. Free plan available at https://cloudlinker.eu/register.
Install the CloudLinker agent on each computer with a local printer.

Compatibility
-------------
Odoo 16, 17, 18 — Community & Enterprise.
